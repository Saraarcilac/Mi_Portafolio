Proyectos

Proyecto 1 — SMIG

El propósito es desarrollar una aplicación móvil para el monitoreo y gestión de ganado en tiempo real para ser utilizada tanto por ganaderos a pequeña escala, como por empresas productoras de lácteos o carnes. El sistema permitirá a los usuarios monitorear diversas variables del ganado, como el peso, la edad, la nutrición, las enfermedades, la ubicación en tiempo real, entre otras. Además, la aplicación incluirá alertas y notificaciones sobre eventos importantes relacionados con el ganado, como cambios de ubicación, problemas de salud o fechas de eventos clave como el parto. El sistema también podrá escalar de acuerdo con las necesidades de los usuarios, desde pequeñas fincas hasta grandes empresas productoras.

Tecnologías: HTML, CSS, Java

Mi rol: Diseñadora y desarrolladora
