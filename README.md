# Portfolio - Sara Arcila

Este es mi portafolio personal, creado como parte del proyecto final.

## Secciones incluidas
- **About Me:** presentación, educación y habilidades.
- **Projects:** tres proyectos con descripción, tecnologías y capturas.
- **Hobbies:** intereses personales.
- **Contact:** información de contacto y formulario funcional (Formspree).

## Cómo ejecutarlo localmente

1. Descarga o clona este repositorio.
2. Abre la carpeta en Visual Studio Code.
3. Abre el archivo `index.html` en tu navegador.  
   (O usa la extensión **Live Server** para vista en tiempo real.)
